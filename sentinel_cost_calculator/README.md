# Sentinel Cost Calculator\n\n## Description\nThis repository contains an interactive web-based Sentinel cost calculator for estimating costs based on user-defined inputs.\n\n## Folder Structure\n- `sentinel_cost_calculator/`\n  - `frontend/`\n    - `components/` - Folder for reusable React components (Form, Charts, Results)\n    - `services/` - Folder for API integrations with backend services\n    - `App.js` - Main React app (entry point)\n    - `index.js` - Initial React entry point\n\n  - `backend/`\n    - `app.py` - FastAPI main server application (route definitions and control logic)\n    - `azure_api.py` - Backend integration for fetching usage from Azure APIs\n    - `pricing_model.py` - Pricing rules and logic for calculating licenses, credits, and cost breakdowns\n    - `test_data.json` - Mock usage data file for testing backend development\n\n  - `README.md` - Instructions for setting up and running the tool\n  - `requirements.txt` - Python dependencies for FastAPI backend.
